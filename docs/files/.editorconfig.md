# .editorconfig

## What

editor 共通設定 (`root = true`)。UTF-8 / LF / 2-space (Python は 4-space) / trim trailing whitespace (Markdown は除外) / insert final newline。

## Why

human + AI が混在編集するリポジトリで、tab/space や line-ending の drift で PR diff が汚れることを防ぐ軽量規律。CI で強制はしない (lint 系に委ねる)。

## How (usage)

editor (VS Code / JetBrains 等) が起動時に読み込み、buffer 設定に適用。

## Constraints

- なし (CI 強制対象外)

## Change impact

- indent_size 変更 → 既存全ファイルへの影響大

## Audience-specific notes

### For AI agents
- 役割タグ: `editor-config`, `whitespace-discipline`

### For human engineers (新卒レベル)
- VS Code 等の editor が自動適用 — 意識しなくて良い

### For third parties
- editor governance の最低限実装
