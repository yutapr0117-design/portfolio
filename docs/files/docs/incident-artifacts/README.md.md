# docs/incident-artifacts/README.md

## What

incident-artifacts/ ディレクトリの inventory。24 件の artifact (decision-* / improvement-notes-* / archived workflow) を 3 セクションで列挙。

## Why

物理移動なしで inventory 化 (Plan D)。新規 artifact 追加時の README 更新忘れを Check 75 で機械強制。

## Constraints

- **Check 75**: 配下全 artifact を本 README が列挙

## Change impact

- 新規 artifact 追加 → README §2/§3/§4 のどれかに追記


## How (usage)

過去判断・学びの参照源。新規 session 復帰時に文脈再構成のために読む。
