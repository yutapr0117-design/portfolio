# .gitignore

## What

Git の追跡対象除外設定。`node_modules/` / `test-results/` / `playwright-report/` / `__pycache__/` / `.DS_Store` / `.claude/settings.local.json` 等の generated / personal files を除外。

## Why

依存パッケージ / 生成物 / 個人設定 が git 追跡対象に混入することを防ぐ。Check 37 が前提とする。

## How (usage)

`git status` / `git add` でこのパターンを除外。

## Constraints

- **Check 37**: 生成物 / cache が追跡対象に混入していない

## Change impact

- パターン削除 → 該当ファイルが意図せず commit される可能性

## Audience-specific notes

### For AI agents
- 役割タグ: `vcs-ignore`, `personal-files-protection`

### For human engineers (新卒レベル)
- 「commit しちゃダメなもの」を列挙
- 新規 generated dir を作ったら追加

### For third parties
- 健全な VCS 衛生の基盤
