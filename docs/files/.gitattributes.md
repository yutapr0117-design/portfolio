# .gitattributes

## What

Git attributes 設定。`*.webp` / `*.mp3` 等の binary を `binary` に明示マーク (CRLF 変換 / 自動 diff 抑制)。

## Why

binary AIO metadata (XMP / ID3) を CRLF 変換から保護。`text=auto` の default で誤って LF→CRLF 変換されると XMP / ID3 が破損する可能性。Check 15 (binary asset 保護) が前提とする。

## How (usage)

`git checkout` / `git diff` / `git merge` で gitattributes に従って binary 扱い。

## Constraints

- **Check 15** (architecture-validation step 14): .gitattributes 存在必須
- **WebP/MP3 binary mark**: AIO metadata 保護の前提

## Change impact

- *.webp / *.mp3 マーク削除 → AIO metadata 破損リスク

## Audience-specific notes

### For AI agents
- 役割タグ: `binary-protection`, `crlf-safe`

### For human engineers (新卒レベル)
- ここを触ると binary が壊れる可能性 — 削除や緩和は禁止

### For third parties
- binary AIO metadata 保護の基盤
