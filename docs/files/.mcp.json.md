# .mcp.json

## What

Claude Code project-scope **MCP (Model Context Protocol) config**。現状は `mcpServers: {}` の placeholder。将来 MCP server 追加時に drop-in 可能。

`.well-known/mcp.json` (静的 AIO discovery manifest) とは別物 (こちらは Claude Code 用の **live MCP server config**)。

## Why

Claude Code 起動時に `.mcp.json` を読み MCP server を provision する。現状 server 不要だが、ファイル自体は placeholder として存在することで「将来追加が単一キー編集で済む」可搬性。

## How (usage)

```
Claude Code 起動
  └─ .mcp.json を読む
  └─ mcpServers の各 server を provision
  └─ tool / resource / prompt が agent から呼び出し可能になる
```

## Constraints

- **Check 79**: JSON parse 可能 + mcpServers dict
- Check 23 (一般 JSON 構文)

## Change impact

- MCP server 追加 → secret 設定 + Claude Code 再起動
