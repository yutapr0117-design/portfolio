# .nvmrc

## What

Node Version Manager (nvm) 用 Node バージョン pin。内容は `24` の 1 行。

## Why

local 開発と CI workflow (`node-version: '24'`) を合わせる。`nvm use` で自動切替。`engines.node` (`^20.19.0 || ^22.13.0 || >=24`) との整合は Check 69 が機械強制。

## How (usage)

```
$ nvm use   # .nvmrc を読んで Node 24 に切替
```

CI 上は setup-node@v6 が直接 24 を install (本ファイルは local 開発者向け)。

## Constraints

- **Check 69**: engines.node が `24` を許容

## Change impact

- Node メジャー変更 → package.json engines.node + 全 workflow node-version pin 同期
