# robots.txt for AI-Driven PM Portfolio
# Public crawl policy applies to both search and AI crawlers.
# Additional machine-readable context is provided via llms.txt and llms-full.txt.
#
# ══════════════════════════════════════════════════════════════════
# CRAWLER STRATEGY — 3-TIER DESIGN
# ══════════════════════════════════════════════════════════════════
#
# Tier 1 — Real-time search / answer generation bots:
#   Allow without delay (maximize citation speed)
#   These serve live user queries; fast access = higher citation probability.
#
# Tier 2 — User-instructed fetch bots:
#   Allow without delay (seamless UX when user asks AI to read the site)
#
# Tier 3 — Foundation model training / scraping bots:
#   Allow with Crawl-delay (resource protection while permitting training)
#
# ══════════════════════════════════════════════════════════════════
# CRAWL BUDGET PRIORITY
# ══════════════════════════════════════════════════════════════════
#
# /llms.txt and /llms-full.txt are the highest-value resources for AI crawlers.
# These files contain the authoritative context that prevents AI hallucination
# about this project. Crawl budget should be concentrated here.
#
# ══════════════════════════════════════════════════════════════════
# EXCLUDED FILES
# ══════════════════════════════════════════════════════════════════
#
# /googlea7059bedc6fe8bdc.html is a Google Search Console ownership verification
# file. It has no semantic value for AI crawlers or users and is excluded
# to concentrate crawl budget on high-value AIO resources.
