This repository operates under **C5 (Human Writes Zero Code)** — all implementation code is generated by AI under direction of the human PM (Yuta Yokoi). The design, prompts, constraints, audits, and merge decisions are exclusively the human PM's work; implementation is the AI's work. External code contributions would violate C5's clarity of authorship and contradict the proof-of-work nature of this portfolio.

The seven hard constraints are:
- **C1 Boring Technology** — zero external frameworks/libraries in the shipped site (no React/Vue/Svelte/Tailwind/Bootstrap/Framer Motion)
- **C2 IIFE** — main logic lives inside IIFEs; no global-scope pollution
- **C3 ErrorBoundary** — View Transition API errors handled by an explicit error boundary
- **C4 No Framework Re-proposal** — frameworks are permanently rejected
- **C5 Human Writes Zero Code** — human writes design + prompts only; all implementation code is AI-generated
- **C6 AIO Integrity** — AIO published layer (`llms*.txt`, `.well-known/*`, JSON-LD, binary metadata) must not change without orchestrator approval
- **C7 KARTE CDN SRI Non-Application** — no SRI on KARTE CDN
